``traceroute`` field reference
------------------------------

.. list-table::
   :header-rows: 1
   :class: longtable
   :widths: 1 3

   * - Field (Type)
     - Description

   * - ``ts`` (time)
     - Timestamp

   * - ``src`` (string - addr)
     - Address initiating the traceroute.

   * - ``dst`` (string - addr)
     - Destination address of the traceroute.

   * - ``proto`` (string)
     - Protocol used for the traceroute.
